# jq macos arm

Apple Silicon (m1/m2) compatible jq binary

## download
```bash
curl -fsSL 'https://github.com/ciiqr/jq-macos-arm/releases/latest/download/jq' -o 'jq'
chmod +x 'jq'
./jq --version
```
